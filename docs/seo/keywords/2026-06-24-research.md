# Keyword research: WebMCP.md (agentic-web / AEO niche) — 2026-06-24

**Product:** webmcp.md — EN agent-instruction site for the WebMCP standard, monetised by the
$49 **WebMCP Agent Skill** (a drop-in SKILL.md that teaches any agent to discover + call WebMCP
tools on a live site).

**Positioning lifted from the live copy:** "The new SEO — but for AI agents." Audience = AI
agents + the developers who deploy them. Core technical surface: imperative API
(`navigator.modelContext`), declarative API (HTML `<form>`), tool discovery
(`/.well-known/web-mcp` + in-page manifest), JSON Schema args, Chrome 149 origin trial,
agentic browsing / Gemini in Chrome.

**Existing pages / anchors (mapping targets):**
- `/` home with anchors `/#what`, `/#agents`, `/#skill`, `/#pricing`, `/#self-demo`
- `/skill` — agent documentation ("How to operate a WebMCP site")
- `/about`
- machine endpoints: `/.well-known/web-mcp`, `/api/mcp`, `/llms.txt`, `/sitemap.xml`

**Language:** EN primary. NL ceded to webmcp.nl — only break into NL where uniquely valuable
(see NL note at the bottom; recommendation = skip for now).

**Strategic implication:** the buyer (a dev who needs to make their site agent-operable, or an
agent-builder who wants a ready skill) is reached through the **education + how-to funnel**, not
through "buy" queries. Pricing/skill pages are conversion endpoints, not SEO landing pages.

6. **`web model context protocol`** → `/#what`. Low-comp full-name synonym of the high-comp head term.

## Mapping summary (every keyword has an owner)

- **Optimize existing:** home `/#what`, `/#skill`, `/#self-demo`, `/#pricing`, and `/skill` carry
  the bulk of fundamentals + how-to + transactional terms. No keyword left unmapped.
- **New pages needed (2):**
  - `/webmcp-vs-mcp` — anchors the entire comparative cluster (highest content ROI; competitors own it, we don't).
  - `/what-is-webmcp` (long-form pillar) — optional but lets us compete on `webmcp explained` and host FAQ schema beyond the home anchor.
- **New angle on existing page:** adopt "agentic engine optimization" + "seo for ai agents" + "agent-ready website" language explicitly on `/#what` / `/#skill`.

## NL note (webmcp.nl owns Dutch)

Recommendation: **skip NL for now.** The Dutch agentic-web search volume is a fraction of EN,
webmcp.nl owns the language, and EN content budget has far higher ROI in this niche (the spec,
the docs, and the assistants are all English-first). Only revisit if a uniquely Dutch query
emerges (e.g. a NL regulatory/banking agent-readiness angle) — none surfaced in this research.
